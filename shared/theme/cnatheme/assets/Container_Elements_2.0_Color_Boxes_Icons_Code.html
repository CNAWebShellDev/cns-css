Color Boxes with Icons

<div class="color-box">
<div class="blue-box spherical">

The icons associated with the colored boxes below are independent from the colors and can be interchanged with any colored box by changing the icon class.

<div class="info-tab blue-box-icon further-reading-icon" title="Further Readings">

Here are the different icon classes:

 
further-reading-icon
extend-learning-icon
did-know-icon
audio-icon
video-icon
try-it-icon
tip-icon
website-icon
note-icon
safety-icon
tools-icon
 

 

NOTE:

  • The title of each box (FURTHER READINGS, EXTEND YOUR LEARNING, etc.) is enclosed in <strong></strong> tags. This tag provides the following attributes:
    • font-weight: 900; text-transform: uppercase; letter-spacing: .02rem; and inherit the color of the box being used.
  • Using <strong></strong> tags with other text elsewhere in the box will have the same attributes above.
  • Ensure you change the title="" to reflect the icon used, if using a different icon with a different color.
