Code Syntax highlighting

To provide high-quality syntax highlighting for any markup language/code follow these steps.

  1. Insert the following code in your HTML content page to display the black Code Block for inserting your code you want to highlight.

    <pre>
    <code></code>
    </pre>

  2. Add the language CSS class you want to display to the <code> tag.

    • <code class="xml">

  1. Next, copy and paste your working block of code you want highlighted and displayed inside the black Code Block. See the HTML/XML sample below.

HTML, XML Sample

<pre>
<code class="xml">

<!DOCTYPE html>
<title>Title</title>

NOTES:

  1. Ensure the following code is inserted in your HTML page.

    • To the Top of the Page in the <head></head>

      • <link rel="stylesheet" href="/shared/theme/highlights/styles/vs2015.css">

      • <script src="/shared/theme/highlights/highlight.pack.js"></script>

    • To the Bottom of the Page


      • <script>hljs.initHighlightingOnLoad();</script>

  2. Depending on the language you are working with you may need to indent the code in some areas.

Highlighting Specific Code Snippets

To highlight specific snippets of code from a block of code, follow these steps.

  1. Identify the specific snippet of code you want highlighted.

  2. Next, wrap this specific snippet of code with another starting <code class="php">  tag and a closing </code> tag. Include the language class in the starting code tag, <code class="php"> .

    IMPORTANT:

    • If the snippet of code you're highlighting is located in the middle of a block of code, ensure to close the <code class="php"> </code> tag above. (See PHP sample below.)

    • If there is a block of code below the highlighted code, ensure to wrap the remaining code with <code class="php"> </code> tags. (See PHP sample below.)

  3. Then, add the css class highlight to the <code class="php"> tag. The highlight class will wrap the snippet of code in an outlined red box.

    • <code class="php highlight">

PHP Sample

<pre>
<code class="php">
function mychildtheme_enqueue_styles() {
$parent_style = 'parent-style';
</code>

<code class="php highlight">
wp_enqueue_style( $parent_style, get_template_directory_uri() . '/style.css' );
</code>

<code class="php">
wp_enqueue_style( 'child-style', get_stylesheet_directory_uri() . '/style.css', array( $parent_style ) ); } add_action( 'wp_enqueue_scripts', 'mychildtheme_enqueue_styles' );
</code>
</pre>
