Paragraph

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur.

<!-- BEGINNING Paragragh -->
<p>Insert Text Here!</p>
<!-- END Paragragh -->

NOTE: The <p> tag defines a paragraph.

Source

Source: Instructor Resources, "Assess Your Learning" by Nelson, a division of Thomson Canada Ltd.

<!-- BEGINNING Source -->
<p class="source"><strong>Source:</strong> <em>Instructor Resources, "Assess Your Learning" by Nelson, a division of Thomson Canada Ltd.</em></p>
<!-- END Source -->

NOTE: Use class="source" inside opening <p> only.

Section

Typically used on the Course Evaluation page

There will be three (3) assignments is this course, with a combined value of 30% of the final course grade. These assignments are practical in nature, and will require you to demonstrate your knowledge of the associated course topics. Completed assignments will be submitted for grading through the Dropbox on or before the date indicated in the course schedule.

<!-- BEGINNING Section - used on the Course Evaluation page -->
<section>
<p>Insert Text Here</p>
</section>
<!-- END Section - used on the Course Evaluation page -->

NOTE: Use only <p></p> inside section tags. Add multiple <p></p> if required.

Links

This is an Internal Link.

<!-- BEGINNING Internal Links -->
<p>This is an <a href="#">Internal Link Text Here!</a>.</p>
<!-- END Internal Links -->

NOTE: Replace the # with the actual internal link.


This is an External Link opens in new window.

<!-- BEGINNING External Links -->
<p>This is an <a href="https://www.google.ca/" target="_blank">External Link Text Here!<span class="sr-only"> opens in new window</span><span class="glyphicon glyphicon-new-window" aria-hidden="true"></span></a>.</p>
<!-- END External Links -->

NOTE: Replace the https://www.google.ca/ with the actual external link.
The class="sr-only" located in the <span class="sr-only"> opens in new window</span> is used to hide information that is only intended for screen readers.
The class="glyphicon glyphicon-new-window" located in the <span class="glyphicon glyphicon-new-window" aria-hidden="true"></span> displays the icon.


This is a Download Link download pdf document.

<!-- BEGINNING Download Links -->
<p>This is a <a role="button" href="#/" target="_blank" id="download-link-tt-top" class="tooltips-link" data-toggle="tooltip" data-placement="top" title="" data-original-title="(PDF ##KB)">Download Link Text Here!<span class="sr-only">download pdf document</span><span class="glyphicon glyphicon-download-alt" aria-hidden="true"></span></a>.</p>
<!-- END Download Links -->

NOTE: Replace the # with the actual download link.
The class="sr-only" located in the <span class="sr-only"> opens in new window</span> is used to hide information that is only intended for screen readers.
The class="glyphicon glyphicon-new-window" located in the <span class="glyphicon glyphicon-new-window" aria-hidden="true"></span> displays the icon.
