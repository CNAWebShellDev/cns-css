NOTE: This is a 4-column responsive table. Additional columns and rows can be inserted using a html editor or manually (if you're familiar with the code). The responsive design of this table does not allow the current view to be displayed on smaller electronic devices. Therefore, it is EXTREMELY IMPORTANT to follow these procedures:

1. When resizing columns please ensure that the width is contained inside the table heading tags ONLY<th width="10%">, NOT in the <tr> or <td> tags.

2. On smaller electronic devices the column headings are NOT displayed. Therefore, it is important to update and include the data-th="" in all <td> tags with the name of the column it resides in.

For Assistive Technologies it is important to add the following elements:

1. Use the <caption> element to identify the table in a document. This is particularly useful for screen-reader users browsing the web page in “table mode” where they can navigate from table to table.

2. If the table has both column and row headings (th), the scope attribute should be used. The scope attribute can be set to row (scope="row" ) or col (scope="col" ) to denote that a header applies to the entire row or column, respectively. The same principle applies to header cells spanning multiple rows (scope="rowgroup" ) and columns (scope="colgroup" ).
