Captions for All Responsive Images

<!-- BEGINNING - Responsive Image w/ Text Wrap and Caption (400px Width) -->
<figure class="pull-right img-400"> <img src="https://via.placeholder.com/400x200" class="img-responsive" alt="Insert Alternative Text Here!">
<figcaption>Insert Figure Caption Here! <a href="#">Insert Link Text Here!<span class="sr-only"> opens in new window</span><span class="glyphicon glyphicon-new-window" aria-hidden="true"></span></a>.</figcaption>
</figure>
<!-- END - Responsive Image w/ Text Wrap and Caption (400px Width) -->

NOTE: Use the <figcaption></figcaption> tags to provide descriptive text on your images. Use the <a href="#"> to provide a link to further information about the image. If the image has no caption remove the <figcaption> </figcaption> tags and all content in between.

The class="sr-only" located in the <span class="sr-only"> opens in new window</span> is used to hide information that is only intended for screen readers.
The class="glyphicon glyphicon-new-window" located in the <span class="glyphicon glyphicon-new-window" aria-hidden="true"></span> displays the icon.
